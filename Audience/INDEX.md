# Audience/

This folder is where you build the relationship with your audience in public. It holds newsletter drafts and sent archives, social-media posts (threads, LinkedIn posts, short-form video scripts), podcast episode outlines and show notes, and any long-form content like essays or tutorials. The goal is a single place where you can see your full publishing history, spot what resonates, and never start from a blank page — every new piece can reference or repurpose something that already landed.

## How to use

1. **Draft each newsletter issue** in its own file; mark status in frontmatter (`draft → scheduled → sent`).
2. **Write social content in batches** — one file per batch of 5–10 posts, tagged by platform and topic.
3. **For each podcast episode**, create one file with: guest name (if any), outline, key questions, show notes, and transcript link after recording.
4. **After publishing**, add a brief performance note (open rate, replies, likes, shares) to the bottom of the file so you have benchmarks.
5. **Maintain `topics-backlog.md`** — a running list of ideas with rough priority so you never stare at a blank content calendar.

## File-naming convention

```
2026-MM-DD-aud-{type}-{slug}.md
```
Examples:
- `2026-05-21-aud-newsletter-issue-14.md`
- `2026-05-21-aud-social-batch-twitter-may-w3.md`
- `2026-05-21-aud-podcast-ep-07-guest-name.md`
- `topics-backlog.md` (evergreen)
