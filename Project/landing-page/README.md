# rails-starter

A blank-slate starter for full-stack apps with **Rails 8 + Inertia.js + React 19 + PostgreSQL**.

## Quick start

```bash
bin/setup      # installs gems, creates and migrates the database
npm install    # installs JS dependencies
bin/dev        # starts Rails (:3000) + Vite (:3036)
```

Requires Ruby 3.3.6, Node 20.19+ (or 22.12+), and PostgreSQL 10+.

See `CLAUDE.md` for the full guide — what's inside, how to add pages, the design system, SSR, deployment, and more.
